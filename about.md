# Acerca de Paperboy — La Portería

Paperboy es un webcomic que combina humor, tecnología y observaciones de la vida cotidiana. Creado con amor y dedicación, cada tira busca arrancar una sonrisa y ofrecer una perspectiva fresca sobre el mundo que nos rodea.

## El Proyecto

Este sitio está construido con Jekyll y desplegado en GitHub Pages. El código fuente está disponible en [GitHub](https://github.com/albertomaydayjhondoe/Porteria).

## Tecnologías Utilizadas

- **Jekyll**: Generador de sitios estáticos
- **GitHub Pages**: Hosting gratuito
- **Docker**: Para desarrollo local
- **CSS Grid y Flexbox**: Para un diseño responsive
- **Vanilla JavaScript**: Para interactividad
